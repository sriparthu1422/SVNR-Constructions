# SVNR Constructions

This repository contains the source code for the SVNR Constructions website.

## Project Structure

The project is divided into two main parts:

-   **frontend/**: The React-based user interface.
-   **backend/**: The server-side logic (Node.js, Express, Socket.io, MongoDB).

## 🚀 Live Deployments

- **Frontend Website:** [https://svnr-frontend.vercel.app](https://svnr-frontend.vercel.app)
- **Admin Dashboard:** [https://svnr-admin.vercel.app](https://svnr-admin.vercel.app)
- **Backend API:** [https://svnr-backend.onrender.com](https://svnr-backend.onrender.com)

## Getting Started (Local Development)

### Frontend
1.  Navigate to the directory: `cd frontend`
2.  Install dependencies: `npm install`
3.  Start the development server: `npm run dev`

### Admin Panel
1.  Navigate to the directory: `cd admin`
2.  Install dependencies: `npm install`
3.  Start the development server: `npm run dev`

### Backend
1.  Navigate to the directory: `cd backend`
2.  Install dependencies: `npm install`
3.  Ensure your `.env` is configured with `MONGODB_URI` and `JWT_SECRET`.
4.  Start the development server: `npm run dev`
